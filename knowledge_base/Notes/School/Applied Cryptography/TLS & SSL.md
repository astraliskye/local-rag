# Certificates on the Web
i.e. how do you know you're connecting to the correct website?

ECC and [RSA]() for the public key algorithms
- Google uses ECC
Browsers come with a long list of CAs that the browser will trust
Can see certificate information for each website as well as the entire keystore of the browser
