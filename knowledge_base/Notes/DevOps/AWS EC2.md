# AWS EC2
Amazon Web Services Elastic Compute Cloud. **Compute!!!**
## AMI
Contains the OS, application server, and applications
## Instance type
Hardware
- CPU
- RAM
## Tags
Key-value pairs that help with managing EC2 instances and creating policies for managing and accessing those instances
## Key pair
Public/private keypairs for securely accessing instances from OpenSSH (.pem) or PuTTY (.ppk)
